【常见问题】

Q: 我想快速开始？
A: 运行 python cli.py，输入 HELP 查看命令

Q: 数据会丢失吗？
A: 不会，只要运行了 SAVE 命令。启动时会自动加载。

Q: 支持重名吗？
A: 支持，只要电话号码不同。

Q: 可以导入导出吗？
A: 使用 SAVE/LOAD 功能，数据以 JSON 格式存储

Q: 性能如何？
A: 10,000条记录下，查询时间 <10ms，查看详情运行 test_performance.py
